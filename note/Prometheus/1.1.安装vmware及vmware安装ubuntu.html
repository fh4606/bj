2、修改虚拟机网络

为了和我讲课的服务器ip保持一致，需要修改下虚拟机的ip网段。点击编辑--虚拟网络编辑器，如下图：

选择nat模式 ，子网ip修改为：192.168.11.0 应用，确定

3、创建虚拟机

桌面双击打开vmware workstation pro，点击创建新的虚拟机，如下图：

默认，下一步

选择“稍后安装操作系统”--点下一步

选择"linux",版本选择 Ubuntu64 位，下一步

虚拟机名称默认，位置可以修改成非c盘。点下一步

磁盘默认20G，修改成50G（测试够了，生产还是在弄大点）。下一步

完成

编辑虚拟机，光驱里面 使用ios镜像文件“ubuntu-20.04.4-live-server-amd64.iso”

启动虚拟机。

三、安装ubuntu20.04

1、设置语言

默认--回车

2、设置键盘

默认--选择"Done",然后按回车

3、配置网络

a、dhcp自动获取ip

默认是dhcp获取ip地址

b、配置静态ip

通过上下方向键，把光标移动到ens33 这行，然后按“回车” ，回来的框里选择”Edit IPv4“回车，如下图：

弹出编辑 ens33 ipv4配置，默认选择automatic dhcp，按回车后，出来的菜单中选择“Manual”，如下图：

Manual配置，按下图填写。（ip地址，网关根据实际修改）填完后，移动方向键，选择save，按回车后保存

4、配置代理

默认为空。选择“Done”，回车

5、配置软件源

默认是ubuntu的官方源，这里先默认，后面进入到系统我们在修改为阿里云源。选择“Done” 回车

6、磁盘配置

a、默认分区（和自定义分区二选一）

默认，使用方向键把光标移动到“Done”，然后回车

磁盘分区信息：

默认，选择Done 回车

确认是否继续，继续后会格式话磁盘。

选择continue继续

b、自定义分区（和默认分区二选一）

选择free space按回车，在选择Add GPT Partition如下图：

创建swap分区，我设置swap分区大小为4G，根据实际修改

创建/boot分区

再次选择free space按回车，在选择Add GPT Partition，大小设置1G，格式为ext4，挂载/boot，如下图：

创建/分区

再次选择free space按回车，在选择Add GPT Partition，大小设置为空（为所有剩余大小），格式为ext4，Mount为/，如下图：

确认下分区大小是否正确，没问题后选择done在回车

提示格式化硬盘，选择continue（继续）--回车

7、资料设置

输入服务器的用户名，主机名，密码，确认密码。

8、Enable ubuntu advantage

默认，选择"Done"，回车

9、是否安装SSH Server

光标在install OpenSSH server按空格键，表示选中，install OpenSSH server选中后（【X】），按方向键移动到Done，按回车

10、安装应用组件

默认（最小化安装），方向键移动到"Done"，回车

开始安装。安装完成后，选择”now reboot“回车。重启后，全部完成。

四、问题

开启虚拟机，出现【此主机支持Intel VT-x，但Intel VT-x处于禁用状态】的提示。

解决：

进入BIOS，将【Intel Virtual Technology】设为【Enabled】保存退出即可
